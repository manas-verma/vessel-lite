# Vessel Lite

## Introduction
A quick backend server to test out the [Vessel](https://www.vessel.dev/) API.

## Installation
1. Clone the repo
2. Run `npm install`

## Usage
1. Add the following variables to the `.env` file:
```
PORT=8080
VESSEL_API_KEY=<your API key>
VESSEL_ACCESS_TOKEN=<your access token>
```
2. Run `npm start`
